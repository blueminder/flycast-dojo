Ways To Play

Using Match Codes & Relay Servers, Flycast Dojo can start a P2P game session behind firewalls, so that you can play against others without the need for a Virtual LAN or matchmaking service. Just exchange codes with your opponent and start your game!
Bundled in Fightcade, the leading online matchmaking service for retro games.

To keep up with the latest Flycast Dojo prereleases, extra features and additional Lua scripts, check out the FC+ Installer (Windows, Linux).
Bundled in Arkadyzja, a new matchmaking platform, initially created for DuckStation multiplayer, that provides matchmaking and a slick game management interface running on top of Flycast Dojo's existing P2P & Relay features.
