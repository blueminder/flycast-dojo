experimental::basic_concurrent_channel::try_receive

Try to receive a message without blocking. 

template<
    typename Handler>
bool try_receive(
    Handler && handler);

Fails if the buffer is full and there are no waiting receive operations.

Return Value

true on success, false on failure.
